this project made with flutter using Cubit, Firebase Authintication, Firebase Firestore, FirebaseStorage

## Features

* Add-Romove Product
* Show Categores
* Show ProductList
* Show Developer Data
* About Application

## Installation
### 1. Clone the repo
```
git clone https://github.com/MohammedAttia3104/iti_final_project.git
cd iti_final_project
```
### 2. Run pub get
```
flutter pub get
```
### 3. Run the app
```
flutter run
```
